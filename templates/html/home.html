This application has two basic parts. The REST API and the Web Interface. The Web Interface for Detection and Recognition of Objects in Images and Videos can be accessed through this link. Info on how to use the API is given below.

RESTful API


How to Use the API

The API can be called like shown in the python code snippet below. Something similar can also be done using other languages. 

            		import requests
			API_URL = 'http://35.196.101.87:8080/obj_det_api'
			IMG_PATH = '/home/frankle/something/something/image_folder/some_image.jpg'
			files = {'file': open(IMG_PATH, 'rb')}
			r = requests.post(API_URL, files=files)
			print(r.json())

If all the operations are performed successfully, the above code snippet will produce a response which will look something like the following: 

            		{
            			'predictions': {'bicycle': '70.86 %', 'person': '63.84 %'}, 
            			'message': 'Photo Uploaded and Predictions Made', 
            			'status': 'success'
            		}
